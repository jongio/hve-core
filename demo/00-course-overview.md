# Mastering HVE Core: The Complete Course
## A Pluralsight-Style Video Course on Hypervelocity Engineering

### Course Overview

HVE Core is Microsoft's open-source prompt engineering library that transforms GitHub Copilot into a constraint-based engineering workflow system. This course takes you from zero to mastery across 20 modules organized in 4 semesters.

### Semesters

| Semester | Name | Modules | Focus |
|----------|------|---------|-------|
| 1 | Foundations | 1-5 | What HVE is, installation, core concepts, first workflow |
| 2 | Core Components | 6-11 | Agents, instructions, prompts, skills, collections, validation |
| 3 | Building and Customizing | 12-16 | Custom artifacts, design thinking, security, project planning |
| 4 | Mastery | 17-20 | Supply chain security, CI/CD, enterprise scaling, contributing |

### Target Audience

Engineers who build, ship, or manage software and want to use GitHub Copilot with structured, validated workflows. No prior HVE experience required. Familiarity with VS Code and GitHub Copilot is assumed.

### Estimated Duration

20 modules at 5-12 minutes each. Total: approximately 3 hours.
